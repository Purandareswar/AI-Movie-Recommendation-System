# 🎬 Content-Based AI Movie Recommendation System

## Overview
An AI-powered movie recommendation system built using Python, Streamlit, and Machine Learning.

## Features
- 5000+ Movie Dataset
- Content-Based Filtering
- Cosine Similarity Algorithm
- Interactive Streamlit UI
- Movie Genre Recommendations
- Clickable Movie Search Links

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Streamlit

## Machine Learning Technique
Content-Based Filtering using CountVectorizer and Cosine Similarity.
